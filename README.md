- Баланс монет расходуется при выполнении игровых действий

### Сохранение данных

## Инвентарь

### Функционал

- Общее количество слотов: 50
- Сетка: 5 слотов в ряду, список прокручиваемый
- По умолчанию инвентарь пуст
- Изначально доступно 15 слотов
- Оставшиеся слоты разблокируются по порядку за монеты (нельзя разблокировать следующий слот, пока не разблокирован предыдущий)
- Стоимость разблокировки каждого слота настраивается
- Считает общий вес предметов

### Слоты

- Разблокировка происходит при нажатии на слот при достаточном количестве монет
- После разблокировки слота, слот сразу становится доступным
- Слот может содержать предмет или стак предметов
- Если количество предметов в слоте равно 1, значение стака не отображается

### Сохранение данных

- Сохраняется содержимое инвентаря и количество предметов в стаке
- Сохраняется расположение предметов в слотах
- Сохраняется состояние разблокированных слотов

### Опционально

- Реализовать перетаскивание предметов (drag-and-drop) между слотами
- Реализовать объединение (стакание) предметов при перетаскивании между слотами (для предметов у которых MaxStack выше 1)
- Реализовать обмен предметов местами при перетаскивании между слотами
- При нажатии на предмет отображать всплывающее окно с полной информацией о нём

## Gameplay

### UI

- Собрать UI по референсу

### Кнопка “Добавить монеты”

- Добавляет случайное количество монет (от 9 до 99)
- Выводит в консоль сообщение `Добавлено ([количество]) монет`

### Кнопка “Добавить предмет”

- Добавляет 1 случайный предмет (weapon, head, torso) в инвентарь
- Выводит в консоль сообщение `Добавлено [предмет] в слот: [id слота]`
- Если нет свободных слотов, выводит ошибку `Инвентарь полон`

### Кнопка “Добавить патроны”

- Добавляет случайное количество патронов (от 10 до 30) случайного типа (PistolAmmo, AssaultRifleAmmo) в инвентарь
- Если в инвентаре уже есть патроны данного типа, сначала пополнить стак в найденном слоте, если останется остаток, его добавить в следующий слот с не полным стаком или отправить в пустой слот
- Выводит в консоль сообщение `Добавлено ([количество]) [тип патронов] в слот: [id слота]`
- Если нет свободных слотов, выводит ошибку `Инвентарь полон`

### Кнопка “Выстрел”

- Делает выстрел из рандомного оружия в инвентаре
- Выводит в консоль сообщение `Выстрел из [оружие], патроны: [тип патронов], урон: [урон]`
- Если нет патронов, выводит ошибку `Нет патронов для [оружие]`
- Если нет оружия, выводит ошибку `Нет оружия`

### Кнопка “Удалить предмет”

- Удаляет предмет или стак предметов в случайном слоте
- Пустые и заблокированные слоты не учитываются
- Выводит в консоль сообщение `Удалено ([количество]) [предмет] из слота: [id слота]`
- Если все слоты пустые, выводит ошибку `Инвентарь пуст`

### Информация

- Текущее количество монет отображается в интерфейсе
- Текущее количество монет обновляется при изменении баланса
- Текущее вес отображается в интерфейсе
- Текущее вес обновляется при изменении количества предметов
- После нажатия, на любую из кнопок, визуал инвентаря должен быть актуализирован
